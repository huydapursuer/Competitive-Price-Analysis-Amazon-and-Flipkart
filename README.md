# Competitive-Price-Analysis-Amazon-and-Flipkart
This is my project on analyzing the competitive landscape of Amazon and Flipkart regarding of the book pricing. In this project, I practice my Power Pivot, Power Query and PowerBI skill alot.

You can see the dashboard here, however it will be expired after 12 days: https://app.powerbi.com/groups/me/reports/fd856679-03f0-4eb1-b4a3-801026d07007/871bcfeae7baa1007923?experience=power-bi

The dataset from: https://www.kaggle.com/datasets/mandan/amazon-vs-flipkart-book-prices

Firstly, I clean the data using Power Query and Practicing analyze it using Power Pivot.
Secondly, I use Power BI to visualize key metrics, which can be useful for monitoring, understanding the price strategy of each platform.
